# 🚗 Smart Parking System

A scalable backend application built using Java and Spring Boot.

The project focuses on building a production-ready Smart Parking System with secure authentication, parking slot management, Redis-based waiting queue, Kafka event streaming, email notifications, PDF receipt generation, and Docker support.

---

## 📌 Features

- Secure JWT Authentication & Authorization
- Parking Slot Management (CRUD)
- Vehicle Registration & Management
- Slot Booking & Release
- Waiting Queue using Redis
- Automatic Slot Allocation
- Parking Fee Calculation
- Dashboard APIs
- Real-time Dashboard Updates using WebSocket
- Event-driven Architecture with Apache Kafka
- Email Notifications on Vehicle Entry & Exit
- PDF Receipt Generation
- Swagger API Documentation
- Docker Support
- PostgreSQL Database Integration
